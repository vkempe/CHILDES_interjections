# CHILDES Interjections

Analysis of caregiver and child interjection use across English CHILDES corpora.

## Contents
- `CHILDES_interjections.Rmd` — main analysis script
- `.rds` files — preprocessed data objects

## Requirements
R packages: tidyverse, childesr, lme4, lmerTest

## Usage
Load the `.rds` files and knit the Rmd. No re-extraction needed.
